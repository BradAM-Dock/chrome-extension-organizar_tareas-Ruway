# chrome-extension-organizar_tareas-Ruway
- Ruway es un gestor de tareas simple pero poderoso integrado directamente en tu navegador. Captura ideas al vuelo, organiza tus prioridades diarias y tacha tus pendientes de la lista sin perder la concentración. La herramienta perfecta para estudiantes, profesionales y cualquier persona que quiera hacer más en menos tiempo.
